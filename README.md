### Installation
Make sure you already installed a package management tool [dep](https://golang.github.io/dep/docs/installation.html)

Clone source code:
```
$ cd $GOPATH/src
$ mkdir -p github.com/kdakean
$ cd $_ && git clone https://your-forkrepo/kdakean.git
```

Run an application:
```
$ dep ensure
$ ./watch.sh
```
Now you good to go...
